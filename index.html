File mới — dữ liệu chưa được load
Để tiếp tục làm việc với dự án cũ, hãy tải file .html (có data) hoặc .json backup mà bạn đã lưu trước đó.

Nhập trang từ người dùng khác
Người dùng khác dùng app → viết tài liệu → Lưu project → gửi file .json cho bạn → bạn chọn trang muốn merge vào project của mình
